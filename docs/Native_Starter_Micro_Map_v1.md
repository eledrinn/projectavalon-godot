## KEY LEGEND

| Symbol | Meaning |
|--------|---------|
| `~` | Shallow water/wetland (wadeable, slow movement) |
| `·` | Path/traversal connection |
| `│ ┌ ┐ └ ┘ ═ ║ ╤ ╧ ╥ ╨` | Elevation boundaries/structures |
| `○` | Active breathing hole (hazard) |
| `SAFE PATH` | Safe route through hazard |
| `[UPPERCASE]` | Major location/landmark |

## LOCATION BREAKDOWN

### 1. SPAWN POINT — Hearthroot
**Grid Position:** Southwest quadrant  
**Coordinates (relative):** 25m E, 25m N from SW corner  
**Key Features:**
- Root overhang creates natural shelter
- Dry ground pocket (first safe space)
- East-facing spawn orientation
- Immediate climbable root tutorial

### 2. MAIN PATH — Central Thoroughfare
**Route:** Hearthroot → Breathing Holes → Shellmound → Woven Arch  
**Length:** ~120m  
**Traversal Time:** ~90 seconds (wading pace)  
**Design Notes:**
- Curved path (prevents straight-line sprinting)
- Natural chokepoint at Breathing Holes
- Visibility progression: roots → open water → mound → arch

### 3. QUESTGIVER — Shellkeeper Elder
**Grid Position:** Central-north  
**NPC Placement:** On Shellmound summit  
**Interaction Zone:** 3m radius around mound top  
**Quest Trigger:** "The Departure Rite" (tutorial quest)  
**Return Loop:** Quest completion requires returning to Woven Arch clearing

### 4. ENEMY POCKET — Strider Ledge
**Grid Position:** East-central, Tier 2 elevation  
**Composition:** 3-5 Strider-class fauna  
**Aggression:** Low (territorial, not pursuit)  
**Placement:** Guards climbing route to Waymarker Vista  
**Tactical Options:**
- Fight: Limited space favors melee
- Sneak: Water approach muffles sound
- Climb: Bypass via alternative root path

### 5. HAZARD POCKET — Breathing Holes
**Grid Position:** South-central depression  
**Dimensions:** 12m x 8m  
**Hazard Points:** 5 active vents  
**Safe Path:** 2m wide timing-based route  
**Cadence:** 4s dormant → 1s warning → 3s active → 2s dissipate  
**Reward:** Shortcut to Tier 2 Ledge (bypasses main climb)

### 6. LANDMARK — Woven Arch
**Grid Position:** Northeast corner  
**Function:** Departure Rite ritual site  
**Features:**
- Village boundary marker
- Ritual clearing (6m diameter)
- Three marker stones
- Invisible transition wall (future expansion gate)

### 7. RETURN LOOP
**Primary Loop:** Spawn → Shellmound → Woven Arch → (complete rite) → Shellmound → Spawn  
**Optional Extended Loop:** Include Waymarker Vista before return  
**Loop Time:** 5-7 minutes (exploration), 3 minutes (return)

### 8. FUTURE WORLD VISTA — Waymarker Point
**Grid Position:** Eastern edge, Tier 3 elevation  
**Access:** Requires full climb sequence (Hearthroot → Tier 2 → Vista)  
**View:** Distant Foothill/Redwood massif  
**Function:** Promise of world beyond cradle  
**Unlock:** Optional fast-travel point

## CRITICAL PATH SUMMARY

**Shortest Path (Speedrun):**  
Hearthroot → Breathing Holes (safe path) → Shellmound → Woven Arch  
*Time: ~3 minutes*

**Intended First Play:**  
Hearthroot → Explore → Breathing Holes (learn hazard) → Tier 2 Ledge (combat) → Waymarker Vista (vista moment) → Shellmound (quest) → Woven Arch (completion)  
*Time: ~7 minutes*

**Completion Requirements:**
- Speak to Shellkeeper Elder
- Reach Woven Arch clearing
- Witness distant vista (optional but encouraged)
- Return to Elder for Departure Rite blessing

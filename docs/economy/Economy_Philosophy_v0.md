## Guiding Principles (Tentative)

1. **Place matters** — Resources should have regional identity
2. **Exploration rewards** — Gathering should encourage travel and discovery
3. **Class integration** — Economy systems should reinforce class fantasy where possible
4. **Friction with purpose** — Trade limitations should create meaning, not tedium
5. **Long-term viability** — Systems should support MMO-style longevity without requiring full implementation in the vertical slice

## Deferred Decisions

The following are explicitly deferred beyond the 1–10 vertical slice:
- Full auction house implementation
- Advanced crafting professions
- Guild/corporate economic structures
- Cross-region trade routes
- Player housing/economy tie-ins
